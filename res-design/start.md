# 开始使用

## 初步下载 & 运行

首先，前往[此处](/support/download)下载最新版本的 Hidden Browser 

由于文件较大，只能使用夸克网盘进行分享。建议先转存至网盘之后再用客户端下载。

下载完成后，你应该会得到一个 HiddenBrowser-V_1_0_0.zip 请将其解压。

> 文件名末尾的版本号会随时改变

解压后得到的文件像这样:

```
HiddenBrowser-V_1_0_0.zip
└-HiddenBrowser
  ├-Main
  ├-简易手册
  ├-启动HiddenBrowser.bat
  ├-启动HiddenBrowser以兼容模式.bat
  └-软件官网
```

运行 `启动HiddenBrowser.bat` 便可以打开浏览器啦。

如果遇到问题（如不出现窗口、缺失 dll），请前往[常见问题解答](/support/docs/faq)寻求帮助

## 使用

软件打开后，你应该能注意到在右上角有一个汉堡菜单（“三”）这是软件的主菜单，与 Hidden Browser 相关的操作需要在这里进行

> 注意，汉堡菜单是紫色的，紧贴于窗口关闭按钮下方，而非 Bing 搜索或其他网站中的汉堡菜单。如图
